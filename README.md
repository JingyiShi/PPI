# PPI

To cite:

* Shi, J., Zheng, M., Yao, L. and Ge, Y., 2018, June. A publication-based popularity index (ppi) for healthcare dataset ranking. In 2018 IEEE International Conference on Healthcare Informatics (ICHI) (pp. 247-254). IEEE.
